## ♡ Device Design Philosophy

This version is **not a mobile-only build**.

It is intentionally designed to feel **best on laptop screens**, where the layout has more room to breathe and the emotional presentation feels richer. At the same time, it still adapts well for mobile users, so the website does not break the experience when opened on a phone.

### Device intent summary

| Device Type | Intent |
|---|---|
| **Laptop / Desktop** | Primary design target |
| **Tablet** | Comfortable secondary experience |
| **Mobile** | Supported and visually pleasant, but not the main composition target |

### What “laptop-first” means here

- more spacious horizontal layout behavior
- richer hero presentation
- stronger section breathing room
- better cinematic balance between text and visuals
- styling choices that feel more premium on wider screens
